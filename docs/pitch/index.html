The Problem

Existing Frameworks

  • JUCE — C++, partial RT safety, no lock-free guarantee
  • SuperCollider — RT but no visual editor
  • VCV Rack — visual but not lock-free
  • Pure Data — 30-year-old C codebase

Real-Time Violations

  • Heap allocation in audio callbacks
  • Mutex locks causing priority inversion
  • Unbounded loops and recursion
  • No compile-time RT safety enforcement

A single missed 1.33 ms deadline produces an audible glitch.
Current tools rely on programmer discipline — not compiler guarantees.

The Solution

🦀 Rust Ownership

Compiler enforces RT safety. No allocation, no locks, no data races — at compile time, not runtime.

⚡ Lock-Free Core

SPSC ring buffer for control→audio communication. Generational arena for O(1) node allocation.

🎛️ Visual Editor

React Flow node editor with real-time WebSocket bridge. Drag, connect, and automate — live.

Zero XRuns

Deterministic execution order. Bounded mutation cost. No surprises in the audio thread.

State Continuity

Node replacement transfers oscillator phase, filter state, envelope stage — no clicks.

Plugin Ready

CLAP plugin template included. NIH-plug wrapper. Ships as a DAW instrument.
